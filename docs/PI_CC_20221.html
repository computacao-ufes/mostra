SpotiPI

João Felipe Gobeti Calenzani e Philipe Aguiar Marques

O SpotiPI é uma aplicação WEB para visualizar as principais informações disponibilizadas pela API do Spotify de forma simples e moderna. Ele permite que você tenha acesso a listagem de playlists, além de um ranking de frequência das músicas e artistas mais escutados e a possibilidade de ver fatias de cada gênero em diversas faixas de tempo. Por fim é possível visualizar se o usuário logado é premium além de outras informações pessoais.

PlannYear

Jordan Gonçalves Pereira

PlannYear é um sistema cujo objetivo é a criação de tarefas atreladas, ou não, a um dia visando auxiliar o usuário no seu planejamento de afazeres, possuindo também a possibilidade de registro de gastos realizados.

Renter - Gerência de locações

Gabriel Ferrari Cipriano e Julio Bissoli

O Renter é uma plataforma que visa prover serviços de gerência e administração de locação de imóveis pessoal, por meio de um aplicativo móvel.

HookGator

Elciney Júnior e Leonardo Fontes

O HookGator é um programa que monitora a criação de um processo especificado e injeta nele uma DLL que é responsável por aplicar inline patches nas chamadas de funções da API do Windows.

código no github

eFarm

Gabriel Porto Santos Peixoto

Trata-se de um sistema básico de pesquisa voltado à área farmacêutica, onde clientes podem pesquisar por produtos, escolher dentre algumas farmácias que o possuem, e entrar em contato com o estabelecimento para realização da venda. Proprietários podem cadastrar seus estabelecimentos a fim de obter maior alcance digitalmente.

código no github

ForYou

Amanda Brito Apolinário e Paula Um Porto de Azeredo Bremenkamp

O propósito do sistema é permitir que o usuário acompanhe seus ganhos e gastos mensalmente, podendo adicionar valores de compras, salários, pagamentos, entre outros. O usuário também pode visualizar essas informações em categorias separadas no extrato, e em um gráfico de rosca geral. Além disso, é possível navegar pelos meses anteriores para ver as movimentações financeiras já realizadas.

código no github

CineList

Bruno Menegaz Acerbi e Gustavo Dutra Santolin

O CineList se trata de uma aplicação web com o intuito de disponibilizar ao usuário a possibilidade de organizar filmes que pretende assistir, como também os que já assistiu, por meio do uso de listas interativas. Para isso, o projeto conta com a exibição de um catálogo onde o usuário poderá visualizar vários filmes, assim como dados sobre cada filme, com a opção de adicioná-lo à lista desejada.

código no github

Traveller Help

Bruno Gomes de Azevedo

O aplicativo Traveller Help tem como objetivo tornar a vida do usuário mais organizada proporcionando a ele um aplicativo de fácil visualização dos gastos em outras moedas sem ser o real. Nele consegue visualizar data, valor da moeda no momento da compra e quanto foi o gasto em real. Possuindo também uma aba de fazer o cálculo da cotação das moedas, sem precisar cadastrar um gasto.

código no github

Chess Visualization

Ezequiel Demetras Silva

Chess visualization é uma plataforma onde enxadristas de qualquer modalidade podem realizar treinos de tática, com o detalhe de as peças não estarem presentes no tabuleiro, forçando o jogador a montar o tabuleiro mentalmente para a partir dai resolver o problema.

código no github

Meu Evento Favorito

Gustavo Brunoro Nahuz, Henrycky Wottikosky da Fonceca, Julia Caldeira Mendonça

O projeto Meu Evento Favorito consiste numa plataforma para gerenciamento de Eventos. O contexto é de uma empresa que faz esse gerenciamento, portanto, não diferentes usuários acessando, apenas a equipe responsável. É possível cadastrar eventos com muitas informações, excluir os eventos permanentemente e marcá-los como concluídos. Os eventos cadastrados persistem no banco de dados até que sejam excluídos.

código no github

MyPoké

Lucas Mendonça Emery Cade, Thiago Damasceno Silva

O sistema MyPoké é um sistema de gerenciamento de bancos de dados de pokémons voltado para pequenos ou grandes campeonatos. Ele consta de uma base de dados de todos os pokémons da primeira geração e um sistema de cadastro, edição e visualização das equipes cadastradas com gráficos que facilitam a análise do campeonato para partes interessadas.

código no github

Caronapp

Lorenzo Miniguite Scarmaussa e Henrick Bartolo

Caronapp é um aplicativo desenvolvido em Flutter que tem como objetivo ser um facilitador para pessoas que frequentemente viajam e optam por realizar ou participar de caronas seja por questões de comodidade, falta de opções, ou financeiras. O sistema possibilita que o usuário ofereça, pesquise e participe de caronas, bem como entre em grupos com perfis de caronas específicas, oferecendo uma plataforma centralizada.

código no github

Memento Studio

Hericles Bruno Koelher e Mikaella Ferreira da Silva

O Memento Studio, do latim “Lembre-se de Estudar”, é um aplicativo de flashcards que visa auxiliar estudantes em suas jornadas de estudo. A ideia é que o aplicativo permita que os usuários criem/busquem baralhos e possam utilizá-los para treinar/reforçar suas memórias acerca de um conteúdo alvo. De posse de um baralho, qualquer usuário retentor de uma conta em nossa plataforma poderá compartilhar sua criação com outros usuários.

código no github

HelpingPaw

Christian Junji Litzinger State e Leonardo Franco Emerick Albergaria

Pensando na problemática do abandono e maus tratos com animais, nós criamos o HelpingPaw: Um hub que visa promover a interação entre seus usuários, permitindo-os expor casos relacionados com a temática. Nele você pode visualizar os posts de outros usuários, assim como as informações de contato de quem criou o post, filtrar os posts por filtro de tipo de post e tipo de animal, criar um post, etc.

Alerta do Tesouro

Atílio Antônio Dadalto e Henrique Coutinho Layber

O Alerta do Tesouro consiste em um site com taxas dos títulos do Tesouro Direto que permite ao usuário criar notificações acerca delas para ser alertado via e-mail, sendo personalizadamente notificado sobre mudanças em taxas de títulos escolhidos do Tesouro Direto, sem a necessidade de ficar vigiando estas taxas manualmente.

Sistema de Vendas e Estoque (SISVE)

Elaine Dias Pires, Filipe Gomes Arante de Souza e Juliana Camilo Repossi

O SISVE é uma aplicação web de interface facilitada para vendas direcionada a pequenos empreendedores. Identificamos que diversos autônomos de pequeno e médio porte apresentam dificuldades em administrar seus estoques, suas vendas e ter conhecimento básico do seu empreendimento. Tendo em vista esses desafios, pensamos em implementar um sistema de forma a resolver esses impasses administrativos.

código no github

Alumni

Débora Ghiotti, Kevin Vieira e Luísa Rios.

A Plataforma Web Alumni foi criada com o objetivo de auxiliar o aluno universitário a planejar e organizar todos os aspectos relacionados à conclusão da sua graduação. Com essa ferramenta, o aluno tem acesso facilitado e dinâmico a todas as informações acerca do seu progresso acadêmico, disciplinas obrigatórias e optativas, horas de extensão, atividades complementares e estágio supervisionado.

SpotiPI

Gabriel Braga Ladislau e Marcus Louriçal Neves Filho

O aplicativo ReadMore incentiva a leitura e a criação do hábito de leitura por meio da organização de seus livros. Com recursos de controle de progresso, o aplicativo funciona como um marcador de páginas digital. Além disso, você pode adicionar livros já lidos, em leitura e livros que deseja ler à sua estante digital. Utilize a função de marcador de página para ter controle do progresso de cada livro.

Mercado Moderno

Daniel Siqueira de Oliveira e Lucas Gabriel de Oliveira Costa

O Mercado Moderno é uma aplicação Web de um mercado online, onde é possível configurar uma loja online e cadastrar produtos para que outras pessoas possam fazer suas compras online.

código no github

Market Price $can

Eduarda Magesk e Letícia Rodrigues

O aplicativo "Market Price $can" tem como objetivo ajudar os usuários a organizar suas compras e economizar dinheiro. Com recursos como criação de listas personalizadas, escaneamento de notas fiscais e compartilhamento com amigos, o app torna o processo de compra mais eficiente. Além disso, é possível comparar preços entre supermercados, facilitando a escolha das melhores ofertas.

código no github

Fang and Friends

João Ubaldo e Vinícius Risso

Fang and Friends é uma versão multiplayer do clássico "Snake game", onde além de evitar as paredes e o próprio corpo, você ainda tem que evitar bater no seu oponente!

código no github

TypeFaster

Carlos Battisti Filho e Gustavo Fim Beloti

TypeFaster é um jogo/teste de digitação com o objetivo de auxiliar os usuários a aprimorar sua velocidade e precisão na digitação, com foco na codificação. Para começar, o usuário terá a opção de escolher a linguagem de programação na qual deseja praticar. Os trechos de código serão gerados por meio da OpenAI API.

código no github

UniVerse

Rhuan Garcia de Assis Teixeira

O UniVerse é uma rede social baseada em texto, inspirada no Twitter, CreekSecrets e Fizz Social. Com uma interface intuitiva, os usuários podem publicar posts, interagir com curtidas e comentários, e explorar notificações de engajamento. O projeto utiliza tecnologias como Java, Spring Boot, React e MongoDB.

código no github

Muraliz-E

Matheus Santos de Moura

O Muraliz-E é um aplicativo voltado para auxiliar o contato de estudantes acerca de assuntos de disciplinas da faculdade. O estudante pode postar suas solicitações no mural e outros estudantes que tem a disciplina referente a solicitação cadastrada serão notificados para que possam auxilia-lo.

MoneyMinder

Lucas Giovani de Paula Salgado

O MoneyMinder é um sistema de assistente financeiro para registros de pagamentos, permite registrar pagamentos parcelas. Possui função de exibição dos registros em um calendário, além de possui um dashboard com algumas métricas para auxiliar no acompanhamento da vida financeira.

BoomBlitz

Lucas Pereira Taborda e Renzo Henrique Guzzo Leão

BoomBlitz é um jogo de navegador inspirado na sequência de jogos Bomberman e seus derivados. Nesta versão você é um dinossauro que tem de escapar de monstros??? Encontre a chave, desvie de seus inimigos e escape da invasão de monstros neste jogo de navegador.

código no github

Let's Cook!

Guilherme Silveira Gomes Brotto e Nilo Garcia Monteiro

Let's Cook! é um aplicativo mobile que permite aos usuários buscar receitas com base em ingredientes e nutrientes escolhidos, além de dietas específicas caso desejem seguir. As receitas são exibidas para o usuário que pode visualizar os ingredientes necessários, o modo de preparo e receitas similares de cada uma. O objetivo é auxiliar quem deseja cozinhar com os ingredientes disponíveis em sua casa, mas estão sem ideias.

código no github

Eugor

Pedro Igor Gomes de Morais e Matheus Saick de Martin

Eugor é um jogo em estilo rogue-like desenvolvido em Python. Uma das principais características de Eugor é sua geração procedural de mapas, a qual permite o player ter sempre uma aventura diferente a cada run. O objetivo do jogo é sobreviver, avançando de fases, para isso o jogador deve eliminar todos os fantasmas que o perseguem a cada dungeon misteriosa que adentra. Fuja, lute, sobreviva somente se for capaz em Eugor!

código no github

NutriPlanner

Marlon Moratti do Amaral e Vinicius Nunes Pereira

O objetivo do sistema é auxiliar em dietas. Para isso, conta com a dinâmica de metas diárias de macronutrientes a serem consumidos. As metas são estabelecidas dadas as métricas biológicas fornecidas pelo usuário. No sistema, o usuário cria uma biblioteca própria com alimentos e receitas, especificando seus macronutrientes e, através deles, registra sua alimentação ao longo do dia.

código no github

Risk of Survivors

Antonio Lucio Braga Secchin, Bruno Lopes Altoé e Vitor Facco Calmon

Risk-of-Survivors é um platformer 2D com elementos de roguelike. O jogador é colocado em um mundo em que pode andar, correr, pular e deve enfrentar hordas de criaturas com sua arma, se tornando melhor com o tempo ao ganhar experiência. O objetivo final de Risk of Survivors é sobreviver até o tempo terminar e escapar do planeta em que o jogo é ambientado.

código no github

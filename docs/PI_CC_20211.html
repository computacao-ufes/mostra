Katify

Sophie Dilhon Gama, Beatriz Matias Santana Maia

Katify é um software desenvolvido com o objetivo de ajudar no controle de gestão pessoal usando quadros Kanban. Através dele é possível dividir as atividades de um projeto em cartões, junto de suas descrições e tags personalizadas. Com o sistema, o usuário pode categorizar as atividades de acordo com a etapa que ele se encontra, como, não inicializado, fazendo, completado e descartado.

Price Action

Alex Santos de Oliveira, Abraão Jesus dos Santos, Miguel Almeida Carlini

O projeto Price Action baseia-se em uma plataforma com intuito de facilitar a vizualização de gráficos de ativos da bolsa de valores. O site disponibiliza gráficos de linha e de candleStick em intervalos de tempo de 20 dias, semanas e meses. Os dados apresentados provém de uma API chamada Alpha Vantage com dados atualizados diariamente. O front-end da aplicação foi desenhado com bootstrap e gráficos do google charts.

Gaming Hub

Danielo Erler Lima, Enzo Baioco Cussuol

O GamingHub é uma plataforma de jogos voltada ao público gamer, que tem como proposta facilitar o acesso a informações sobre os jogos inclusos no catálogo, contendo dados para cada jogo, como: preço, seus desenvolvedores, descrição do jogo, acesso a páginas como metacritic e steam, e uma sessão de comentários única da plataforma.

App To Do

Fernando Azevedo Peres, Igor Mattos dos Santos Varejão

O aplicativo Todo tem como objetivo tornar a vida do usuário mais organizada proporcionando a ele um aplicativo de fácil visualização mas ao mesmo tempo com várias features que permitem ao usuário definir tarefas de forma simples e intuitiva. Além disso possui um design moderno para tornar o uso do app mais confortável para o usuário.

PLgym

Leonardo Deorce Lima de Oliveira, Pedro Victor Alves de Souza Santos

O PLgym é uma ferramenta que facilita e organiza os seus exercícios físicos em casa. Cada usuário é capaz de adicionar em sua lista pessoal os exercícios que desejar dentre os fornecidos no nosso banco de dados. Todos os exercícios possuem um endereço de acesso a um vídeo tutorial (curado pela nossa equipe) sobre como melhor executá-lo.

código no github

GameCenter

Matheus Lenke Coutinho, Igor Wandermurem Dummer

O GameCenter é um aplicativo (Android e iOS) de gerenciamento de jogos, em que o usuário pode registrar quais jogos estão na sua lista e seus status (biblioteca, jogando, etc), além de visualizar detalhes sobre os mesmos, independente de plataforma e conectando com a api da IGDB e Twitch. Foi desenvolvido utilizando React Native, com uma API Rest em Java Spring Boot e banco de dados MongoDB.

Melhor Projeto 1º Lugar

MaterialStock

Renan Moreira Gomes, Ícaro Madalena do Nascimento

O projeto Material Stock é um sistema de lançamento de dados referentes a locação de objetos. Nele é possivel cadastrar materiais, objetos compostos desses mesmos materiais, colaboradores, clientes e ordens de serviço de locação. O intuito é facilitar o registro e a consulta de informações referentes a esses registros.

Moneymoney

Ryan Tavares Farias da Silva Monteiro, Rogério Medeiros dos Santos Jr

Aplicação Web com o objetivo de auxiliar no acompanhamento financeiro pessoal. A aplicação possui funcionalidades como inserção de transações financeiras e visualização, divisão de transações em clusters e visualização dos dados em gráficos. Além ter seus dados salvos e seguros em sua conta na nuvem.

Melhor Projeto 2º Lugar (empate)

App do Café

Vitor Berger Bonella, Luis Eduardo Freire da Câmara

Aplicação web desenvolvida para ajudar a fortalecer o cenário do café no Espirito Santo. No projeto criamos um API para acompanhar o preço da saca do café e fomentar a comercialização de café. A aplicação web permite a criação de contas para que um usuário possa criar anúncios de venda ou compra de sacas de café, além disso permite ele observar o preço atual do café e suas variações.

código no github

Os Filhos das Queimadas

Joana Santos Martins, Eduardo Afonso Ribeiro Rodrigues, Mateus Freitas Couto

O jogo "Os filhos das queimadas", desenvolvido pela nossa equipe é um RPG interativo que conta com uma história totalmente original e cheia de mistérios. Para a história foram construídos cenários próprios compatíveis com o desenvolver da trama. A ideia do jogo é explorar os mapas, combater inimigos e coletar pistas para que possa por fim resolver o grande mistério do qual o jogo gira em torno.

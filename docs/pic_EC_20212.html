Carrinho controlado via Wi-Fi

Afonso Magalhães, Diogo Brandão, Thamya Donadia

Carrinho de controle remoto, utilizando o módulo Wi-Fi da placa ESP8266 e operado por uma aplicação web voltada para dispositivos móveis.
git

Painel Stranger Things

Athila Rodrigues, Bernardo Broedel

O projeto recebe uma mensagem do usuário através de um aplicativo de celular, que com o auxílio do Modulo Bluetooth, envia para o arduino que é responsável por acender determinados LED's de acordo com as letras da mensagem enviada.
git

Detector de turbidez com LED

Caio Fiorotti, Henrique Santos, Matheus Schreiber

Composto por um Arduino UNO e um resistor LDR, com o auxílio de uma fonte de luz é possível determinar o nível de turbidez de um copo de água. Além de mais barato que um sensor de turbidez real, o projeto também é mais personalizável.
git

Janela automatizada baseada na chuva

Camila Abreu, Vitor Marques

Um alerta com sensor de chuva. Assim que o sensor detectar a presença de água, o LED vermelho acende e se a janela estiver aberta o buzzer irá apitar com um delay de 2 segundos, só irá parar de apitar quando a janela fechar. Se não houver água no sensor, o LED verde ficará aceso sem nenhum barulho do buzzer.
git

Mão Robótica

Gabriel Gatti, Hiuri Liberato

O projeto consiste na montagem de uma mão que movimenta os dedos de acordo com comandos recebidos via Bluetooth pelo aplicativo no smartphone do usuário. Dentre os comandos, há opções de gestos pré-estabelecidos ou movimentação separada dos dedos.
git

Flanentino

Lucas Alexandre

Desenvolvimento de uma placa compatível com Arduino contendo mais leds e um buzzer.
git

Slot Machine

Daniel F. Alves, Luiz Bazoni, Thiago Lahass

Implementação em Arduino em uma Slot Machine. A máquina consiste em um tipo de máquina de apostas, originalmente americana, baseada em sorteios de ícones e premiações em modas de acordo com as combinações dos ícones sorteados.
git

WALL-ESP

João Brito, Talles Weiller, Vinícius Colle

Um carrinho com uma câmera e com uma garra articulada. Controlado por um ESP32, conseguimos visualizar o que é capturado pela câmera e controlar remotamente os movimentos do carrinho e da garra através de um servidor WEB.
git

DESP LUZ

Bruno Fernandes, João Clevelares, Luca Ávila

O Desp Luz é um projeto de automação residencial, mais especificamente de cortinas e persianas. Estas, por sua vez, serão acionadas automaticamente de acordo com um horário pré agendado através de um smartphone, podendo ser sincronizado com o horário do seu despertador. As cortinas também poderão ser abertas em qualquer horário com um botão ON/OFF.
git

Pitch - RFlyD

Arthur Coelho Estevao, Milena da Silva Mantovanelli

RFlyD - Automação do Inventário em Armazéns. Este projeto tem como objetivo desenvolver uma Vending Machine que aceite uma criptomoeda como pagamento. Espera-se ter uma vending machine física que funcione usando um contrato inteligente e criptomoedas. A vending machine deverá estar integrada com o contrato inteligente de modo que, ao ser feita uma transferência de valor específico, o usuário poderá escolher um produto, de acordo com o valor transferido, e então a máquina deverá liberar o produto e finalizar a operação de venda.
git

Robô Obstáculo

Pedro Henrique Peixoto da Silva, Heron Garcia Lodi e Silva e Wesley Vieira de Moraes

O projeto consiste em um robô autônomo, controlado por Arduino, que deverá percorrer uma determinada área e retornar de volta para o ponto de partida e também será capaz de desviar de obstáculos fazendo para isso uso de um sensor ultrassônico.

Blackjack

Guido Agnezi Deniz, João Vitor Toneto Machado, Miguel Martins Boechat

Usar o arduino para simular um ambiente de jogatina de “Blackjack”, bem como um banco de fichas e sistema de dica para iniciantes.
git

Robô radar

Alexsander Barbosa Moura. Hydrech Urgel Mahoungou Mbouaka.

O objetivo é construir uma Sensor ultrassônico montado sobre um servo motor em cima de um carro, podendo cobrir qualquer direção que o servo motor puder trabalhar
git

Arcade Machine

Fábio Pascoal, Ellyas Santos, Alceu Félix

A ideia central do projeto é construir uma máquina de fliperama simplificada, com três jogos clássicos programados a partir do Arduino Uno, de modo que seja possível alternar a qualquer momento o jogo em execução através do input do usuário.
git

Robô autoequilibrio

Leonardo Roxo, Rafael Marques e Matteo Chiste Trento

Consiste em um robô de duas rodas, projetado para se equilibrar automaticamente por meio de ajustes precisos realizados por um motor de passo, garantindo estabilidade e eficiência no seu funcionamento.
git

Casa Inteligente

João Arthur Subtil, Leonardo Rezende e Lucas Bastos.

É um aparelho que detecta a fumaça no ar, alertando o usuário, e conta com um sensor de presença, que verifica se há alguém no local. O objetivo é trazer segurança e praticidade de forma acessível.
git

Semáforo inteligente

Arthur Sampaio, Jordano Souza, Nathan Freitas

O projeto consiste basicamente em um sistema de semáforos para uma pista de mão única que detecta a presença de pedestres e a intensidade do tráfego, utilizando estas informações para abrir ou fechar o semáforo.
git

Caixa decriptadora

Alexandre Fazzolo, Gabriel Santos, Lilanio Vieira

O projeto integra um site para envio de mensagens criptografadas e uma caixa física que usa biometria e NFC para autenticação e descriptografia. As chaves públicas são geradas na caixa, enquanto as privadas ficam apenas no NFC, garantindo segurança e privacidade.
git

Genius

Enrico Massariol, Matheus Gon Zortéa, Silvio Junior

Nosso projeto será uma recriação do famoso jogo de memória Genius, um jogo divertido e desafiante que testa sua memória.
git

Mão Mímica

Rafaela Capovilla, Rafaela Fernanda e Diana Rosi

O projeto consiste em uma mão impressa em 3D que imita os movimentos da mão do usuário apresentada na webcam, fazendo uso da visão computacional, para identificar os movimentos que a pessoa fez..
git

Painel do Clima

Ana Luisa Casotti, Clarice Curtinhas Santos e Mateus Lizardo

O projeto executado foi um painel informativo das condições climáticas locais usando fitas de Led. A ideia é criar um objeto tridimensional para transmitir dados climáticos sobre o ambiente da UFES. Projeto em parceria com o curso de Arquitetura UFES.
git

Observatório das Árvores

Breno Silva, Felipe Soares, Marcela Fernandes, Raissa Almeida

O projeto consiste em construir uma caixa que visualiza informações de uma árvore. A árvore será identificada por RFID. O projeto é reposável de fornencer dados para o projeto "Painel do Clima" exibir. Projeto em parceria com o curso de Arquitetura UFES.
git

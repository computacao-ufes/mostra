Ardudino

Gabrielly Cariman, Felipe Carpes e Luiz Machado

O projeto consiste em um mini emulador de jogo controlado por um Arduino. O Arduino irá controlar uma tela Oled i2c no qual conterá um jogo baseado no Dino do navegador Chrome (T-Rex Running). Como input do usuário, o projeto conterá dois botões e um acelerômetro.

Jardim Inteligente

André Nemer, Celso Neto, Guilherme Soares, Gustavo Sarti

O projeto tem como função verificar a situação de determinada planta a partir da medição da umidade do solo. Tendo como base esse valor será analisado se a planta precisa de água e, de acordo com as necessidades da mesma, a bomba de água é acionada.
git

Melhores projetos na escolha do público

SpiderBot

Ana Pereira, Felipe Albuquerque, Vitor Gomes

O objetivo é construir uma aranha controlada pelo Arduino Uno, utilizando um Sensor Shield V5.0 e 2 baterias 9V para alimentar 8 servos que irão compor o movimento da aranha. Além disso, utilizaremos o sensor ultrassônico para que a aranha tenha autonomia para desviar de obstáculos próximos.
git

Melhores projetos na escolha do público

Luva VR

Guilherme Ratti, Edgard de Souza T, Matheus Luciano

Criação de uma Luva VR para captar as rotações no espaço tridimensional e replicá-lo em um ambiente virtual no “Unity”, a fim de criar um simples jogo para equilibrar uma bola em uma superfície pelo maior tempo possível com a adição de alguns obstáculos
git

Melhores projetos na escolha do público

Dispensador de medicamentos

Jullie Quadros, Gabriela Belmock e Sofia Sarcinelli

Um dispositivo que contém compartimentos para armazenar doses de remédio, com um menu para a definição cada alarme. Nos horários dos alarmes, abre a seção referente e toca um alarme sonoro até que o usuário aperte o botão indicando que tomou a dose.
git

Marble Maze Labyrinth

João Batista, João Nascente, Pedro Souza e Roger Silva

: O game em questão consiste em uma plataforma suspensa que é controlada por um joystick, movendo-se de acordo com o manuseio do jogador. O objetivo é levar a bolinha de gude até o final do labirinto sem cair nos buracos do caminho, havendo 3 chances.
git

Semáforo inteligente

Arthur Sampaio, Jordano Souza, Nathan Freitas

O projeto consiste basicamente em um sistema de semáforos para uma pista de mão única que detecta a presença de pedestres e a intensidade do tráfego, utilizando estas informações para abrir ou fechar o semáforo.
git

Theremin Ultrassônico

David Leal, Gessica Silva, Renan Duarte

Um instrumento theremin usando um sensor ultrassônico, no qual um auto-falante emite frequências sonoras diferentes em relação a proximidade com o sensor ultrassônico utilizado num raio de 120 cm e controle do volume do alto-falante feito por um potenciômetro.
git

Alarme com laser

José Uliana, Maria Julia Nolasco

O projeto objetiva a construção de um alarme acionado por presença a partir do uso de sensor LDR, um LED, um emissor de som, um teclado matricial e uma placa microcontroladora Arduino Uno.
git

Monitoramento Residencial

Elder Storck, Alan Wehbe, Eduardo Perim, Victor Lucas

Nesse projeto será desenvolvido um sistema de monitoramento residencial. O foco do projeto será a segurança do local. Assim, desenvolveremos um sistema com monitores para abertura de portas e captura de movimentos. Quando detectar algum resultado, enviará uma mensagem para o telegram do usuário cadastrado.
git

Carro com sensor de obstáculos

Camille Policarpo, Gabriel Saymon, Khristyan Vilela e Luana da Ros

Mvimentação autônoma de um carrinho controlado por arduino, sendo o principal diferencial a utilização do sensor HC-SR04 (ultrassônico) para fins de detecção e desvio dos objetos no caminho.
git

Carro de Controle Remoto

Gabriel Pietroluongo, Luiz Bandeira e Otávio Sales

O projeto consiste em construir um carro operado por controle remoto (DualShock 4) onde a comunicação será feita via Wi-Fi. O sistema será embarcado utilizando uma Raspberry Pi, com os dados enviados a partir de um computador utilizando a framework ROS.
git

Robô que desvia de obstáculos

Elias Junior, Fernando Vieira, Aryelle Siqueira, Fodé Sambú

O projeto consiste em um robô que desvia de obstáculos. Inicialmente o robô anda para frente até que encontre um obstáculo. Quando encontra, ele desvia para direita ou para esquerda a depender de qual escolha é a melhor.
git
